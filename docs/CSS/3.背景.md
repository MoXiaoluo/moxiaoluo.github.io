## CSS 的背景样式

使用 background 属性来定义背景样式。

### 背景颜色

background-color

### 背景图片

background-image:url(...)

### 控制背景平铺

background-repeat

| 属性      | 描述           |
| --------- | -------------- |
| no-repeat | 不重复         |
| repeat-x  | 水平重复       |
| repeat-y  | 垂直重复       |
| repeat    | 在两个方向重复 |

### 调整背景图像的大小

background-size:[cover\|contain]

### 背景图像定位

background-position

### 背景附加

background-attachment

### 简写 background

bg-color || bg-image || bg-position [ / bg-size]? || bg-repeat || bg-attachment || bg-origin || bg-clip
